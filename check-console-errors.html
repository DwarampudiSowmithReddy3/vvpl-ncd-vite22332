Open Browser Console (F12) and check for errors

Then navigate to your React app and see what errors appear in the console.

Common React Error Patterns to Look For:

  • Cannot read property 'X' of undefined
  • Cannot read properties of undefined (reading 'X')
  • X is not a function
  • Cannot destructure property 'X' of 'undefined'
  • Uncaught TypeError
  • Uncaught ReferenceError

Instructions:

  1. Open your React app in browser
  2. Press F12 to open Developer Tools
  3. Go to Console tab
  4. Try to navigate to Dashboard/NCD Series/Investors
  5. Copy the exact error message that appears
