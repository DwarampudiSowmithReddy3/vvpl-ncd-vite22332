Check Network Calls

INSTRUCTIONS:

  1. Open your frontend at http://localhost:5178
  2. Open Developer Tools (F12)
  3. Go to Network tab
  4. Clear the network log
  5. Try to toggle a permission
  6. Look for API calls to localhost:8000

What to look for:

  • Do you see a PUT request to http://localhost:8000/permissions/?
  • What is the status code? (200 = success, 4xx/5xx = error)
  • If no request appears, the frontend is not calling the backend
